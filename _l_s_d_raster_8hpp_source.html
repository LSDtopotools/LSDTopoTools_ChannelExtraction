1483 // ///
1484 // /// @details It creates a contour curvature map and identifies channel heads as pixels greater
1485 // /// than a user defined contour curvature threshold value, set by default at 0.1. The threshold curvature
1486 // /// can also be defined as a multiple of the standard deviation of the curvature. Before this function is called
1487 // /// the DEM must be filtered using the wiener filter in the LSDRasterSpectral object in order to remove high frequency
1488 // /// noise.
1489 // ///
1490 // /// Reference: Pelletier (2013) A robust, two-parameter method for the extraction of drainage
1491 // /// networks from high-resolution digital elevation models (DEMs): Evaluation using synthetic and real-world
1492 // /// DEMs, Water Resources Research 49: 1-15
1493 // ///
1494 // /// @param window_radius Integer window radius - suggest 6m.
1495 // /// @param tan_curv_threshold Double curvature threshold value.
1496 // /// @param tan_curv_array 2D array of tangential curvature.
1497 // /// @return LSDIndexRaster of predicted channel head locations.
1498 // /// @author FC, DTM
